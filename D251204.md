##### 변수 
    - 변수 :값을 담는 상자. 값을 재사용하기위해서 줄로 사용
    - 변수 타입 : 상자에 어따한 유형의 값을 담을지 결정
    - 사용방법
    - 타입 명시
    '[타입][변수명] = [담을값]'
    'int age =20;'
    - 타입 추론 : 타입을 적지않고 뒤에 오는 값에 따라 타입이 자동으로 지정됨
    'var [변수명] =[담을값]; //변경 가능'
    'final [변수명] =[담을값]; //변경불가능'
    'const [변수명] =[담을값]; //변경 불가능 => 컴파일 시점 값이 결정되는 변수'
##### 데이터 타입
- 'int' : 정수
- 'double' : 실수(소수점포함)
- 'String' : 문자열 
- 'bool' : 참/거짓
- 'dynamic' : 모든걸 담을수 없는 타입,오류발생할 가능성이 높아짐!
##### 연산자
- '==' 같은지 비교
- '<' 우측이 큰지 비교
- '>' 좌측이 큰지 비교
- '<=' 우측이 크거나 같은지 비교
- '>=' 좌측이 크거나 같은지 비교
- '*' 곱하기
- '/' 나누기
- '%' 나머지
- '~/ 몫
- 'a' 산술연산자 = b' a와b 연산한 결과를 a에 저장
- 'a = b ? c : d' 3항연산자 => a의 값은 b가 참이면 c 거짓이면 d!
- 'a = b ?? c' null 병합연산자 => a의 값은 b가 널이면 c, 아니면 b!
##### 조건문
- 'if(조건식){실행할코드}'
-'if(조건식1){   } else if(조건식2) {} else {}' 조건식 1이 참이
아닐경우 조건식2 비교. 조건식 2도 참이 아닐경우 else 내 증괄호 실행
-'switch' 문도 있음! => 자동완성 활용하기

##### 컬렉션
- 'List<담을 타입> 변수명 = [담을값1,담을값2]' 여러개의 값을 담을때
